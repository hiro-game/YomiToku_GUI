## 🧩 このアプリについて

本アプリは **Microsoft Copilot によって作成された Copilot 製アプリ**です。  
Windows 11 + Python 3.13 で動作確認済みです。

内部で使用する YomiToku が Python 製であるため、  
**パス設定さえ正しく行えば Linux / macOS でも動作可能**です。

## 🛠 使用するランタイム

本アプリは以下のランタイム・ライブラリを使用します。

- **Python 3.10 以上**  
  - Windows / Linux / macOS いずれも対応  
- **PySide6**  
  - GUI フレームワーク（Qt6 ベース）
- **YomiToku（外部ツール）**  
  - 本アプリは YomiToku の実行ファイル（またはスクリプト）を呼び出すだけのフロントエンドです  
  - YomiToku 本体は別途インストールが必要です  
- **標準ライブラリ**  
  - `pathlib`, `subprocess`, `configparser`, `site`, `shutil` など

## ⚙ 設定ファイルの仕様
設定ファイル名はYomiToku_GUI.ini  

### 設定項目
```
save_settings =
```
アプリ終了時に、ユーザーが選択した設定値を保存するかどうかを制御します。  
#####設定値
- 0 … 設定を保存しない(初期値)
- 1 … 設定を保存する  
#####保存される内容
- 入力フォルダ
- 出力フォルダ
- チェックボックスやオプションの状態  
#####動作  
アプリ終了時の設定が YomiToku_GUI.ini に書き込まれます。  
次回起動時の初期値として起動します。  
```
save_log =
```
アプリ終了時に、画面下部のログウィンドウの内容を保存するかどうかを制御します。  
#####設定値
- 0 … ログを保存しない(初期値)
- 1 … ログを保存する  
#######保存される内容
- パス検出ログ
- エラー内容
- YomiToku の実行結果ログ
- その他、画面下部に表示されるすべてのログ  
#######動作  
アプリと同じフォルダに次の形式で保存されます：  
- YomiToku_YYYYMMDD_HHMMSS.log
- ログは毎回新しいファイルとして保存されます
```
yomitoku_path =
```
YomiToku 本体の実行ファイルのパスを指定します。  
#####動作仕様
- 設定ファイルに **有効な `yomitoku_path` がある場合は絶対に上書きしません**  
→ ユーザーが意図的に古いバージョンを使うケースを尊重  
- 設定が空または無効な場合のみ、自動検出を行います  
- `site.getsitepackages()` / `getusersitepackages()`  
- Scripts / bin の探索  
- where / which のフォールバック  
- 設定ファイルは起動時に自動読み込みされます
